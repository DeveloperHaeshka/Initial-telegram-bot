Для начала вам нужно его настроить под себя, и для этого нужно зайти в файл - config.py и заполнить все данные под себя

Для запуска бота вам нужно запустить файл - main.py

Для получение админки вам нужно
1. Запустить бота
2. Найти файл users.db
3. С помощью приложение SQLite открыть его
4. зайти в столбец users, и поменять в параметрах user_status у себя на Rab
